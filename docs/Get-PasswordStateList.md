# Get-PasswordStateList

## SYNOPSIS
Gets all password lists from the API (Only those you have permissions to.)

## SYNTAX

```
Get-PasswordStateList [[-PasswordListID] <Int32>] [[-Searchby] <String>] [[-SearchName] <String>]
 [<CommonParameters>]
```

## DESCRIPTION
Gets all password lists from the API (Only those you have permissions to.)

## EXAMPLES

### EXAMPLE 1
```
Get-PasswordStateList
```
